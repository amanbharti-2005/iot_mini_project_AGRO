# 🌱 IoT Smart Agriculture Rover – AGRO

## 📌 Description
AGRO is a team-based IoT-enabled smart agriculture rover designed for real-time field monitoring. It collects environmental data using sensors, streams live video via ESP32-CAM, and displays insights on a web dashboard for remote decision-making.

## 🚀 Features
- Real-time sensor data monitoring  
- Mobile-based remote control of rover  
- ESP32-CAM live video streaming  
- Cloud dashboards using MQTT & ThingsBoard  
- ML-based object detection (basic implementation)  
- Actuator-based response to environmental conditions  

## 🔧 Sensors Used
- DHT11 (Temperature & Humidity)  
- Soil Moisture Sensor  
- Rain Sensor  
- Light (LDR) Sensor  
- Water Level Sensor  

## 🛠 Tech Stack

### Hardware
- ESP32 / ESP8266  
- ESP32-CAM  
- Sensors & actuator modules  

### Software
- React (Vite)  
- JavaScript, HTML, CSS  
- MQTT Protocol  
- ThingsBoard (Cloud Dashboard)  

## 🧠 Key Highlights
- Integrated hardware + software system  
- Real-time data transmission and visualization  
- Prototype-based development (second iteration with improved performance)
